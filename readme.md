# Auto Typer Pro 
This is an auto-typing tool which simulates actual human typing. 


## Features
- Human Error Logic (Adjustable Threshold)
- Customizable Hotkeys
- Adjustable Speeds 
- Sleek and Compact UI

## Specifications
- Built using Python
- Uses Tkinter for UI
